
# How to download 
```
git clone git@github.com:TASEHana/CD102.git CD102_paper
cd CD102_paper
```
# How to compile

## Download the zipped revtex4 file and copy styles
```
./copyStyle
``` 

## compile with the following script
```
./makePDF main
```

## Output 
The produced PDF file is called main.pdf
